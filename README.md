Ansible ZFS role
================

Install ZFS on Debian Linux.

Requirements
------------

None.

Role Variables
--------------

- `zfs_reboot_on_restart`

Dependencies
------------

None.

Example Playbook
----------------

    - hosts: servers
      roles:
         - { role: HanXHX.zfs }
